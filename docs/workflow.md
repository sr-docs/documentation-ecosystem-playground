Contributors move through workflow slices that expose different parts of the documentation lifecycle.

| Stage   | Purpose                       |
| ------- | ----------------------------- |
| PLAN    | Define what should be created |
| WRITE   | Produce content               |
| REVIEW  | Validate quality              |
| PUBLISH | Deploy changes                |
| OBSERVE | Monitor outcomes              |

| Visitor Sees                      | Underlying Experience    |
| --------------------------------- | ------------------------ |
| How documentation enters a system | Create Content           |
| How quality is maintained         | Review Content           |
| How documentation reaches users   | Publish Content          |
| How documentation evolves         | Improve Content          |
| How the ecosystem improves itself | Documentation Operations |
